# Test

Librerias, SDKs
    - se recomienda el 100% de cobertura
    - Deben ir a la misma posición que he creado mi proyecto
    - Excepción: Si tu proyecto tiene más de 10 mil lineas de código o 30 archivos, es mejor tenerlo en una carpeta test
